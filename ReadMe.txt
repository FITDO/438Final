A game that was created for Advanced Game Development course at Iupui.

This project was built to deminstrate my understanding of collision, scripts, and scenes.

How to Run:

Download the source code from main and use Godot 4.0 to import then compile

or

Download the release for Windows and run the .exe

Created using -

Godot as the game engine

https://sfxr.me/ - was used to create the sounds

Blender was used to create the level and the models in game

Controls -

Move - WASD

Face Direction - move the Mouse

Swing Sword - LeftClick

Fire Arrow - Hold 'E' or RightClick til a red light appears

Dash - Press a shift while holding the direction you want to dash
